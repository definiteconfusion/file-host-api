## Features
- Public and Private File Storage
  - CDN -> Public
  - API -> Private 
- User Account Recovery with Email Tokens
- SQL Injection Protection with Character Screening 
  - SQLite3 Database

## Included Dependencies [Python 3.11]
- Flask
- Etext
